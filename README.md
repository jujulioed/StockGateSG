# Stock Gate

Repositório do Projeto Stock Gate, um sistema de gerenciamento de insumos com o objetivo de reduzir custos de operação de inventário de empresas.

## Tecnologias utilizadas

### Backend

Backend construído em Java SpringBoot, com seu domínimo divido pelas responsabilidades principais da ferramenta: Usuários, itens e notificações, além de outras features relevante que serão implementadas posteriormente.

### Frontend

Frotend construído em Next.js, focado em ter um bom SEO na landing page e foco em performance para boa experiência de usuário.

### Infra

Banco de dados PostgreSQL - rápido, simples e preciso - perfeito para o contexto do projeto

### Containers Docker 

Projeto dockerizado para facil deploy em qualquer VPS
